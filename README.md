~~测试环境：windows   Node.js 18.16.1~~
*测试环境：macOS Sonoma14.5  Node.js v22.2.0*

```js
npm i
```

- win编辑main.js 然后运行批处理即可

或者运行以下命令(win/mac/linux)

```js
node main.js
```
